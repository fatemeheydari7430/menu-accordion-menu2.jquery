گام اول : آشنایی با مفاهیم UI/UX مسلط
طراحی UI مخفف «واسط کاربری» یا User Interface Design است؛ طراحی رابط کاربری، به طراحی زیبایی شناختی تمام عناصر بصری رابط کاربری یک محصول دیجیتال اشاره دارد و "UX" مخفف کلمه User Experience یا «تجربه کاربر» است، تجربه کاربر از برنامه با نحوه تعامل آن‌ها با آن تعیین می شود.
گام دوم : ورود به دنیای برنامه نویسی و آشنایی با مفاهیم پایه در وب
رابط یکنواخت وب بر اساس مجموعه ای بزرگ و رو به رشد از مشخصات است. این مشخصات مفاهیم مشترکی را ایجاد می کند که هرکسی می تواند بر آن بنا کند. یک مثال برای نحوه کار کردن HTTP/1.1 Caching است که 5 فیلد هدر HTTP، 7 کد هشدار HTTP و 12 دستورالعمل حافظه پنهان HTTP را تعریف می کند.Web Concepts مروری بر این مفاهیم و مشخصاتی است که آنها را تعریف می کند. Web Concepts یک نمای کلی ساختاریافته و به هم پیوسته از بسیاری از مفاهیمی که با هم سطح وب را ایجاد می کنند، ارائه می دهد.
گام سوم : HTML5 و CSS3 رو قورت بده
HTML برگرفته از حروف اول عبارت HyperTextMarkupLanguage می باشد و به عنوان یک زبان نشانه گذاری متن ( زبان استاندارد صفحات وب ) می باشد. معمولا در یک صفحه ی موجود بر روی وب مجموعه ای از متن ها، عکس ها، بخش بندی های مختلف وجود دارد، به کمک تگ ها و کدهایی که HTML در اختیارتان قرار می دهد، می توانید یک ساختار مشخص از اطلاعات را در این قسمت ها به وجود آورید و در معرض نمایش بازدیدکنندگان قرار بدهید. Css کوتاه شده یا مخفف عبارت Cascading Style Sheets می باشد که قابلیت صفحه آرایی؛ استایل دهی و زیبا سازی صفحات وب را برای طراحان سایت فراهم می سازد. در واقع ساختار یک صفحه با استفاده از HTML تعیین می شود و سپس با به کار بردن کدهایی دیگر به این ساختار و چیدمان استایل داده می شود. تمامی رنگ ها، ابعاد و اندازه ها، حاشیه ها، مکان قرارگیری، فونت ها و سایر خصوصیات اجزای یک صفحه ی وب، با استفاده از CSS تعیین خواهند شد.
گام چهارم : معرفی GOOGLE DEVELO
دستگاه های اندرویدی روز به روز بیشتر می شوند و اپلیکیشن های ایرانی نیز جای خود را در بازار پیدا کرده اند. ولی متأسفانه محدودیت هایی برای ارائه این اپلیکیشن ها نیز وجود دارد. برای ثبت جهانی این اپلیکیشن ها چه باید کرد؟ شرکت گوگل برای ثبت اپلیکیشن ها در گوگل پلی سرویسی راه اندازی کرده است که بوسیله ی آن می توان اپلیکیشن های اندرویدی را در بازار جهانی به نمایش گذاشت. شما با ارائه اپ خود بر روی گوگل پلی می توانید محصول خود را بفروشید، پشتیبانی کنید و با ارائه ورژن های جدید، آن را آپدیت کنید.
